## Description

- A  todo app using both NestJs and Angular.

## Installation

```bash
$ npm i
```
```bash
create your own .env following .env.example
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
